## Rollback Plan

If issues discovered mid-process:

1. **Before any commits:**
   ```bash
   git reset HEAD  # Unstage everything
   git checkout -- .  # Revert modified files
   git submodule update --init --force  # Reset submodule
   ```

2. **After partial commits:**
   ```bash
   git reset --soft HEAD~N  # Undo last N commits, keep changes staged
   # Or
   git reset --hard HEAD~N  # Discard commits entirely (DESTRUCTIVE)
   ```

3. **Submodule issues:**
   ```bash
   cd infrastructure/templates
   git reset --hard 7a1a5d8  # Force reset to original
   cd ../..
   git submodule update --init
   ```
